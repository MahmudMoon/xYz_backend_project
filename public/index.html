Available Endpoints

  • GET / - Welcome message
  • GET /health - Basic health check
  • GET /health/system - Detailed system information

Network Information Endpoints

  • GET /api/network-info - Get all network info records
  • GET /api/network-info/:id - Get network info by ID
  • POST /api/network-info - Create network info record
  • PUT /api/network-info/:id - Update network info
  • DELETE /api/network-info/:id - Delete network info
  • GET /api/network-info/device/:deviceId - Get by device ID
  • GET /api/network-info/statistics - Get network statistics
  • POST /api/network-info/bulk - Bulk create records
